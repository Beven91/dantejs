## dante

### 一、简介

一个通用库，集成了array,string,date,list,function,number,timer等等的扩展，具体参见文档

可用于浏览器中


### 二、安装

    npm install dantes --save-dev
    
     
### 三、用例
	
    var Dante = require('dantes');

    Dante.string.format('name:{0}','123');

    Dante.date ....
    .....
