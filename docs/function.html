F bind | source:function.js,line:63

名称：返回一个劫持函数，被劫持函数handler调用者转向至getCall调用者，函数柯里化....

函数签名参数：
Pinvoker

调用者 默认为window

Phandler

待调用的函数 注意：在调用时函数中的this指向当前实例

PclosureArgs

额外传递给handler的参数,注意 该参数使用位于handler最后一个 例如：handler(args1,arg2,arg3,...,closureArgs)

E用例：
            
     var Dante = require('dantejs');
     var obj = {name:'hello'};
     var fn = function(arg1,arg2,last_is_context){ alert(arg1+arg2+this.name); }
     var bindFn = Dante.Fn.bind(invoker,fn,{id:2})
     bindFn(1,2); > alert: '12hello';
